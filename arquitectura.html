⚙️ Reglas técnicas clave
📐 Factor /100 en bonos
Bono Soberano, Dólar Linked, Bono CER y Bono Dual cotizan en centavos en el mercado ARS. Para calcular valor real: precio / 100. CEDEARs y Acciones usan precio directo.
💰 Cálculo MEP portafolio
Valor USD de posición = (precio_ars × cantidad) / mep. Se puede usar MEP de compra histórico (mep_compra) o MEP actual del día (AL30 ARS÷USD) para ver ganancia/pérdida.
⚡ Realtime — cómo funciona
El browser se conecta vía WebSocket a Supabase Realtime y suscribe al canal de la tabla mercado. Cada cambio llega sin polling. La evaluación de alertas ocurre inmediatamente en el browser.
🔐 Claves de acceso
Frontend usa anon key (pública, ok para lectura). Scripts Python usan service role key para writes masivos sin restricciones de RLS. Edge Function tiene JWT verification desactivado.
🎨 Sistema de temas
4 temas via data-theme en <html>. Variables CSS: --bg, --text, --green, etc. Tema persiste en localStorage. IBM Plex Mono para valores numéricos.
